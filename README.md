# 🚀 Fintech Innovation Lab – Landing Page

A modern, responsive, and animated landing page built using **Next.js, Tailwind CSS, and Framer Motion**. This project showcases a professional UI with multiple sections like Hero, Features, Testimonials, FAQ, and Contact Form.

## 📌 Features

* ⚡ Fast and optimized with Next.js
* 🎨 Beautiful UI using Tailwind CSS
* 🎥 Smooth animations with Framer Motion
* 📱 Fully responsive design
* 📊 Sections included:

  * Hero Section
  * Stats
  * Partners
  * Features & Benefits
  * Testimonials (Auto Slider)
  * FAQ (Accordion with Tabs)
  * Contact Form
* 🔄 Dynamic state handling using React Hooks

* **Frontend:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Icons:** Lucide React

## ⚠️ Common Issues & Fixes

### Hydration Error (Next.js)

* Caused by browser extensions (like password managers)
* Fix:

  * Run in **Incognito mode**
  * Disable extensions

---

### Images Not Loading

* Ensure images are inside:

```
public/logos/
```

* Access via:

```
/logos/image.png
```
